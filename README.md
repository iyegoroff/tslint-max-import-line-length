## Getting started

`$ npm i tslint-max-import-line-length -D`

## Usage

```json
  // tslint.json
  "extends": [
    "tslint-max-import-line-length"
  ],
  "rules": {
    "max-import-line-length": [true, 100]
    ...
  }
  ...
```
